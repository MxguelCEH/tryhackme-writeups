# Nmap Room (TryHackMe)

## Objective
Learn how to scan and identify open ports using Nmap.

## Tools Used
- Nmap

## Commands
nmap -sS 10.10.10.10
nmap -sV 10.10.10.10

## Key Findings
- Port 22 (SSH) is open
- Port 80 (HTTP) is open

## What I Learned
- How to perform a basic scan
- Difference between stealth scan and version detection

## Notes
- -sS = stealth scan
- -sV = service version detection
